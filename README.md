# ChatFilter

 Filter all bad word for cs2 chat by replacing them with * symbol. The config file is automatic generated in configs/plugins path.
 ```json
{
  "filter_list": ["fuck", "nigger", "niger", "pedo"],
  "ConfigVersion": 1
}
 ```
This word 'fuck' is filter by plugin.
